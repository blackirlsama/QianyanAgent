## 常见问题排查

### 未发现文档
```
⚠️ 未发现任何文档，请检查路径: ...
```
**解决**：检查rag.docs-path配置和文件格式（.md/.txt）

### 文档目录不存在
```
⚠️ 文档目录不存在: ...
```
**解决**：创建目录或修改配置路径

### 召回无结果
```
⚠️ [阶段1-粗排] 向量召回无结果
```
**解决**：检查查询内容或降低minScore阈值
